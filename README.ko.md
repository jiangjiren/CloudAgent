# CloudAgent

CloudAgent is a customized self-hosted AI coding-agent web UI based on the
open-source CloudCLI project.

This localized README is intentionally short so it does not keep outdated
upstream marketing content. The maintained documentation is:

- [English README](README.md)
- [Simplified Chinese README](README.zh-CN.md)

Repository: <https://github.com/jiangjiren/CloudAgent>

Note: some package and command names still come from upstream CloudCLI, such as
`@cloudcli-ai/cloudcli` and `cloudcli`, until the fork is fully renamed.
